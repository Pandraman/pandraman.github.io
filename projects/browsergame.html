    I wanted to make a browser game, just for fun, so when i discorvered pygbag I knew that I hit a gold mine. Changing the game is very easy, you just have to import asyncio and turn the main function into an async function. I followed this tutorial Then to upload it to github pages I had to use github actions. Doing that is not that complicated either: First off all go to the repo settings and allow github actions to read / write files, then use a simple workflow, which i have copied myself from this repository After that you can go to the pages part of settings and then deploy from the gh-pages branch, which should be created by the github actions script. Then enjoy your browsergame. Mine is at: pandraman.github.io/BrowserGame/
